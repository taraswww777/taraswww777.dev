    При работе с GitHub и другими хостингами репозиториев часто приходится копировать URL для <code>git clone</code> — открыть страницу, найти кнопку, скопировать ссылку, переключиться в терминал. Рутина, от которой хотелось избавиться.

    ## Идея плагина

    Мини-плагин для Chrome, который добавляет кнопку для быстрого копирования URL репозитория прямо со страницы. Один клик — и команда <code>git clone</code> с нужным адресом в буфере обмена.

    ## Что сделал

    Первая версия появилась в ноябре 2024 года. Потом были доработки — в <a href="https://github.com/taraswww777/pet-chrome-plugin-git-clone/pull/2" target="_blank">PR #2</a> плагин привели в порядок. Осталось причесать код и опубликовать в Chrome Web Store.

    ## Как это выглядит

    Плагин добавляет кнопку на страницы репозиториев. По клику копируется URL для <code>git clone</code>.
